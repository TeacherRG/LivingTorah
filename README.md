# תורה חיה — Living Torah · Уроки Торы

**Living Torah** — интерактивная платформа для изучения Торы, основанная на учении Ребе — Менахема Менделя Шнеерсона.

---

## О проекте

Living Torah — это лёгкий статический веб-сайт, предоставляющий структурированные уроки по Торе с интерактивными викторинами, поддержкой нескольких языков и настраиваемым внешним видом. Платформа делает учения Хабад-Любавич доступными для русско-, украино- и немецкоязычной аудитории.

**Репозиторий:** [TeacherRG/LivingTorah](https://github.com/TeacherRG/LivingTorah)  
**Контакт:** kosherletter@gmail.com | Telegram: [@chitasclub](https://t.me/chitasclub)

---

## Возможности

- 📖 **Структурированные уроки** — каждый урок разбит на разделы с ключевыми понятиями, цитатами из источников и пояснениями
- 🌍 **Многоязычность** — русский, украинский и немецкий языки с динамическим переключением
- ✅ **Интерактивные викторины** — двухчастные тесты с подсчётом баллов и мгновенной обратной связью
- 🎨 **Темы оформления** — 5 цветовых схем (тёмно-синяя, чёрная, тёмная сепия, светлая сепия, белая)
- 🔤 **Настройка шрифта** — выбор шрифта и размера (Frank Ruhl Libre, Merriweather, Rubik)
- 📊 **Прогресс** — индикатор прохождения урока в верхней части страницы
- 💾 **Сохранение настроек** — предпочтения пользователя сохраняются в localStorage
- 📱 **Адаптивный дизайн** — корректно работает на мобильных устройствах и десктопе
- ⬅️ **Навигация кнопкой «Назад»** — на мобильных устройствах аппаратная кнопка «Назад» перемещает по разделам урока, а не закрывает страницу

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone https://github.com/TeacherRG/LivingTorah.git
cd LivingTorah

# 2. Запустить локальный сервер (любой из вариантов)
python -m http.server 8000
# или
npx http-server

# 3. Открыть в браузере
# http://localhost:8000
```

Или просто откройте `index.html` напрямую в браузере.

Сайт готов к публикации на любом статическом хостинге без дополнительной настройки:
